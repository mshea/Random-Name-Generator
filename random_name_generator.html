Random Fantasy Name Generator

Coming up with good fantasy names is one of the harder tricks for the Lazy Dungeon Master. The following script generates a set of two hundred random names using a mixture of popular yet uncommon first names and last names generated from a pair of words. While many of these names may end up as nonsense, you are encouraged to read through the list until you find a name that clicks for whatever NPC you want to generate.

Random Names - shuffle names

Using This Script

There are many ways to use this script. The easiest is to run it a few times, copying and pasting the results into a single page file you keep in your GM Walk Away Kit. You can also copy a few random names into any cheat sheet you build for whichever game system you choose. You can also keep this page handy and load it on your PC or mobile device whenever you need a good random name for your game.
